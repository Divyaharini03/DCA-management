- npm

### Installation
1. Clone the repository
2. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Running the App
Start the development server from the `frontend` folder:
```bash
npm run dev
```

The application will be available at `http://localhost:3000`.
